# Calvinball League Teams

A dataset containing team information for the Calvinball League.

## Usage

``` r
cb_teams
```

## Format

A tibble with 6 rows and 2 variables:

- team_id:

  Unique team identifier

- team_name:

  Team name (e.g., "Transmogrifiers", "Time Travelers")

## Source

Synthetically generated data inspired by Calvin and Hobbes
